# Training Attendance Analytics Dashboard (Power BI)

## Project Overview

This project presents an interactive Power BI dashboard designed to analyze training participation and attendance trends within an organization. The dashboard provides KPI-based monitoring, team-wise analysis, and managerial performance insights to support data-driven decision-making and improve training engagement tracking.

---

## Objectives

- Monitor overall training attendance using key performance indicators (KPIs)
- Analyze participation trends across teams, roles, and senior managers
- Enable interactive filtering and drill-down analysis for deeper insights
- Support organizational decision-making through clear data visualization

---

## Tools & Technologies

- Power BI Desktop
- Data Modeling
- DAX Measures
- Microsoft Excel (Data Source)

---

## Key Features

- KPI cards for total enrollment, attendance percentage, and absence tracking
- Team-wise and senior manager-wise attendance analysis
- Gender-wise and role-level distribution visualizations
- Interactive slicers and filters for dynamic reporting
- Structured data transformation for accurate analytics
